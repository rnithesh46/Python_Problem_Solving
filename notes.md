# data structure
- list
- tuple
- dictionary
- set

    ### 1)list
    - list is a collection of heterogeneous data
    - mutable
    - allows duplicate
    - num=[],decleration

    ### 2)tuple
    - num=(10,20)
    - a tuple is immutable collection of data

    ### 3)dictionary
    - collection of key-value pair
    - key must be unique but values can be duplicate
    - it is unorder
    - dict={'0':'100'}

    ### 4)set
    - collection of unique values
    - it is mutable
    - no duplicates
    - unordered
    - num={10,20,30}
